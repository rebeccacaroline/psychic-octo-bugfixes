# My Library

##Desription
This is a to-read list. Users can input books they want to read, and check the books off as they're read.

##Dependencies
1. MongoDB
2. jQuery
3. Underscore.js
4. Backbone.js

##Server
1. Clone the repo
2. Execute:
  $ npm install
3. Run:
  $ nodemon index.js
The server will run on port 3000.

##Heroku
To view this app on Heroku, go to mybackbonelibrary.herokuapp.com
